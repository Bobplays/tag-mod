# Tag Mod Launcher

A custom launcher for **Tag** by JetGames with built-in mods and quality-of-life tools.

The launcher runs directly in your browser and is hosted online, so there is nothing to install.

---

# How To Use

1. Open the website.
2. Launch the game.
3. Press `Right Shift` to open the mod menu.
4. Select and toggle your mods.

No installation needed.
